# MiniLife Project Log

## Project Vision
Minilife is a baby simulator game where users control and manage the needs of a virtual baby. The goal is to provide an engaging, beautiful, and interactive experience.

## Development Policies
1.  **Permission First**: No action will be taken without explicit USER permission at every step.
2.  **Rich Aesthetics**: Prioritize modern, premium UI/UX designs (Glassmorphism, smooth gradients).
3.  **Clean Code**: Maintain modular, readable JavaScript and well-structured HTML/CSS.
4.  **Process Tracking**: All significant changes and milestones will be recorded in this log.
5.  **Tech Stack**: Phaser.js for game logic/animations + HTML/CSS for premium UI.
6.  **Cross-Platform**: Designed for Web/Mobile/Desktop using Capacitor wrapping; porting for Consoles.

## Current State
-   Basic project structure established (HTML/CSS/JS).
-   Core stats implemented: Hunger, Sleep, Happiness.
-   Basic interactions: Feed, Sleep, Play.
-   Static baby visual (currently a placeholder `baby.png`).
